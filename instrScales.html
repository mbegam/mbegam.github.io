Contructing a Scale

  1. Use the Mode Button to select the mode.

  2. Line up the "1" on the Degree Circle with the desired mode label on the Mode Ring.

  3. Line up the note on the Pitch Ring on which the scale is to be built with the same mode label.

  4. For each degree of the scale, read the note on the Pitch Ring in line with the white "key" on the Mode Ring.

Examples

D-Major: D, E, F♯, G, A, B, C♯, D

G-Minor: G, A, B♭, C, D, E♭, F, G

B♭-Mixolydian: B♭, C, D, E♭, F, G, A♭, B♭



What's in a Name?

The sequence of notes in a scale must include every letter name exactly once. The only exception is the octave--degree 8--which repeats degree 1, as in "...that will bring us back to Do." For instance, there should never be an E♭ and an E. If both do appear to be indicated, then either the E♭ is really a D♯ or the E is really an F♭. Since enharmonics (different names for the same note) produce the same pitch when played, such considerations are of importance only when reading or writing music--they are meaningless to the listener.
