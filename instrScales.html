Contructing a Scale

  1. Use the Mode Button to select the mode.

  2. Line up the "1" on the Degree Circle with the desired mode label on the Mode Ring.

  3. Line up the note on the Pitch Ring on which the scale is to be built with the same mode label.

  4. For each degree of the scale, read the note on the Pitch Ring in line with the white "key" on the Mode Ring.

Examples

D-Major: D, E, F♯, G, A, B, C♯, D

G-Minor: G, A, B♭, C, D, E♭, F, G

B♭-Mixolydian: B♭, C, D, E♭, F, G, A♭, B♭



What's in a Name?

The sequence of notes in a scale must include every letter name exactly once. The only exception is the octave--degree 8--which repeats degree 1, as in "...that will bring us back to Do." For instance, there should never be an E♭ and an E. If both do appear to be indicated, then either the E♭ must be a D♯ or the E must be an F♭. Since enharmonics (different names for the same note) produce the same pitch when played, such considerations are of importance only when reading or writing music--they are meaningless to the listener.

Further Constraints

A particular scale is an ordered listing of the set of notes that comprise the key of the same name. For example, the D-Major scale shown above lists the notes to be used in a composition written in the key of D-Major. Any accidentals (sharps or flats) found to be necessary when constructing the scale are therefore the same as those indicated in the key signature for that key. A key signature must consist of all flats, all sharps or neither. Combinations of sharps and flats are not allowed, nor are double-sharps or double-flats. These contraints serve to resolve situations such as this:

Is it D♯-Major or E♭-Major?


D♯-Major: D♯, E♯, F♯♯, G♯, A♯, B♯, C♯♯, D♯...Wrong!
E♭-Major: E♭, F, G, A♭, B♭, C, D, E♭...Right
