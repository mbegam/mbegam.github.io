Contructing a Scale

  1. Use the Mode Button to select the mode.

  2. Line up the "1" on the Degree Circle with the desired mode label
    on the Mode Circle.

  3. Line up the note on the Pitch Circle on which the scale is to be
    built with the same mode label.

  4. For each degree, read the note on the Pitch Circle in line with
    the white "key" on the Mode Circle.

Examples

D-Major: D, E, F#, G, A, B, C#, D
