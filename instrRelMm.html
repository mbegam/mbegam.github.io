Identifying Relative Major & Minor Keys

Every key signature applies to two different keys, a major and a minor. Such pairs of keys that share the same signature are said to be relative to each other. To identify the relative minor for a given major:

  1. Use the Mode Button to select "Major".

  2. Line up the note on the Pitch Ring for the major key in question with the "Major" label on the Mode Ring.

  3. Use the Mode Button to select "Minor".

  4. Read the name of the relative minor key from the note on the Pitch Ring lined up with the "Minor" label on the Mode Ring.

Do the same in reverse to identify the relative major for a given minor.
